About Scroll Movie

The Scroll Movie is a component that allows you to scroll through images and display tracks and labels.

Track display timing and labels In addition to options like changing characters, you can also customize your CSS by setting className.

Install

<script src="https://unpkg.com/scroll-movie@1.0.2/dist/main.js"></script>

Usage

When using as vanilla.js


<link rel="stylesheet" href="./css/scroll-movie.css" />

<script src="https://unpkg.com/scroll-movie@1.0.2/dist/main.js"></script>
  
<script>
const tracks = [
  {
    html:
      "<h1>Scroll Movie.js</h1><br />If you scroll, the background image will move.",
    timing: {
      start: 0,
      end: 5000,
    },
    buttonLabel: "animation",
    animation: {
      start: "originalCssStart",
      end: "originalCssEnd",
    },
  },
];
ScrollMovie("#app", {
  tracks: tracks,
  getImage: (index) => {
    const pad = `${index}`.padStart(3, "0");
    return `images/image_${pad}.jpg`;
  },
  imageSize: 758,
  scrollsPerImage: 60,
  sliderBarLength: 60,
  preload: true,
});
</script>
